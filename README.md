# BLX

Simple tool


## Setup

Update packages

```bash
apt update && apt upgrade
```

Install NodeJS

```bash
pkg install nodejs-lts
```


## Usage

Clone the repository

```bash
git clone https://github.com/baetmon/blx.git
```

Change directory

```bash
cd blx
```

Install dependencies

```bash
npm install
```

Run server

```bash
node ./server.js
```
